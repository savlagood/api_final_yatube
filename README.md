# Yatube API

## Описание

API для социальной сети Yatube, позволяющее пользователям создавать публикации, комментировать их, подписываться на других авторов и объединяться в сообщества.

## Функциональность

- Аутентификация по JWT-токену
- Создание, редактирование, удаление и получение публикаций
- Комментирование публикаций
- Подписка на авторов
- Просмотр и фильтрация сообществ
- Пагинация в выдаче публикаций

## Как запустить проект

1. Клонируйте репозиторий и перейдите в него:
2. Создайте и активируйте виртуальное окружение:
```bash
python3 -m venv venv
source venv/bin/activate # для Linux/macOS
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Выполните миграции:
```bash
python manage.py migrate
```

5. Запустите сервер:
```bash
python manage.py runserver
```

## Документация API

После запуска проекта, документация API доступна по адресу:
```
http://127.0.0.1:8000/redoc/
```

## Примеры запросов

### Получение токена

```bash
POST /api/v1/jwt/create/
```
Пример запроса:
```json
{
  "username": "user",
  "password": "password"
}
```
Пример ответа:
```json
{
  "refresh": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
  "access": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
}
```

### Создание публикации

```bash
POST /api/v1/posts/
```
Заголовки:
```
Authorization: Bearer <ваш_токен>
```
Пример запроса:
```json
{
  "text": "Новая публикация",
  "group": 1
}
```

### Получение публикаций с пагинацией

```bash
GET /api/v1/posts/?limit=10&offset=0
```

### Подписка на автора

```bash
POST /api/v1/follow/
```
Заголовки:
```
Authorization: Bearer <ваш_токен>
```
Пример запроса:
```json
{
  "following": "author_username"
}
```

## Права доступа

- Публичный доступ на чтение всех публикаций и комментариев.
- Аутентифицированные пользователи могут создавать публикации, комментарии и подписываться на авторов.
- Изменение и удаление контента доступно только авторам этого контента.
